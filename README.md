# TSS
TOFS TRADER SUPPORT SERVICE 


THIS IS A TRADER SUPPORT SERVICE EXAMPLE HOW TO CONNECT TO THE TSS API TO UPDATE, DELETE AND CREATE ALL KINDS OF DATA (SUCH AS SUPPLEMENTARY DECLARATIONS AND PRODUCTS).

THIS WAS USED TO AUTOMATE A DATA ENTRY PROCESS THAT WOULD OF TOOK 3 MONTHS IF DONE MANUALLY, BUT WAS COMPLETED IN LESS THAN 12 HOURS WITH THIS API SCRIPT.

FURTHER TRADER SUPPORT SERVICE DOCUMENTATION CAN BE FOUND HERE....  https://www.nicustomstradeacademy.co.uk/pdf/tss-declaration-upload-api-guide/
